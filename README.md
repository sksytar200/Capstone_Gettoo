# 우리학교 동아리 웹 플랫폼 GET-TOO
## 📝 프로젝트 소개
우리학교 동아리의 정보를 공유하고 동아리원 모집과 관리를 할 수 있는 사이트입니다.

## 🕰️ 개발 기간
23년 04월 ~ 23년 06월

## 👩🏻‍💻 멤버 구성
- 팀장_박시영
- 팀원_김별
- 팀원_신수현
- 팀원_이규민
- 팀원_김창환

## ⚙️ 개발 환경
- Java 8
- JDK
- IDE: IntelliJ
- Framework: SpringBoot
- Database: Mysql

## ⚒️ 주요기능
로그인 [상세보기]()
- 로그인 시 세션 생성
- ID 찾기, PW 찾기
- DB 값 검증

회원가입
마이페이지
메인페이지
- 동아리 목록 나열

관리자 페이지
- 학생증 확인 및 회원가입 승인/거절

동아리 관리자 페이지
- 가입 신청 승인/거절

동아리 신청 페이지
- 동아리 지원작성 폼
