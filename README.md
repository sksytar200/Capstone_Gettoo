# 우리학교 동아리 웹 플랫폼 GET-TOO

우리학교 동아리의 정보를 공유하고 동아리원 모집과 관리를 할 수 있는 사이트입니다.

## 🕰️ 개발 기간
23년 04월 ~ 23년 06월

## 👩🏻‍💻 멤버 구성
- 팀장_박시영
- 팀원_김별
- 팀원_신수현
- 팀원_이규민
- 팀원_김창환

## ⚙️ 개발 환경
- Java 8
- IDE: IntelliJ
- Framework: SpringBoot
- Database: Mysql
